
# nationbuilder API play

preppin for the unions

## how

### install

```
git clone https://github.com/lizzielashark/nationbuilder API play
cd nationbuilder API play
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
